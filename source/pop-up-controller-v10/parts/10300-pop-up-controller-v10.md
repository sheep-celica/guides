## Card Summary

Main control unit for the Pop-up Controller V10 system and the hub for the accessory connections
used by the rest of the hardware.  

## Details

The Pop-up Controller V10 replaces the factory Light Retractor Relay.  
Additionally, several accessories can be connected to it to allow more customizable pop-up control.

## Fact Cards

- Main role: Main controller unit for the V10 line.
- Fits: T18 Toyota Celica.
- Connections: 12-pin main connector, accessory ports `X1` to `X6` and a 'USB-C'.
