# App Guides

This page groups the desktop app documentation for the Pop-up Controller V10.

## Available Guides

### App Setup Guide

Use this guide if you are installing the desktop app for the first time, preparing the controller connection, or checking that the app can detect the controller correctly.

### App Flashing Guide

Use this guide when you want to update the controller firmware through the desktop app.

### App Usage Guide

Use this guide for day-to-day app tasks such as checking controller information, reading errors, adjusting settings, and deciding when to flash firmware.
