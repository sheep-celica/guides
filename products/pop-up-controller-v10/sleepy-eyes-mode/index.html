How It Works

Sleepy Eyes Mode lets both pop-ups stop at a selected partial-open position instead of fully raising or fully retracting.

Mode overview

Sleepy Eyes Mode uses the dedicated controls connected to X5 to stop both pop-ups at a partial-open target.

When the mode is turned on, both pop-ups first move to UP and then continue to the selected intermediate position.

Default restriction

By default, Sleepy Eyes Mode can only be enabled while the light switch is in the OFF position. That safety behavior can be changed in the desktop app.

While the mode is active

  • The LED under the sleepy eye button stays on.
  • Light-switch commands no longer change pop-up position.
  • The selected rotary position continues to define the target angle.
  • Wink commands still work and return to the same sleepy-eye position afterward.

When the mode is turned off

  • The LED under the sleepy eye button turns off.
  • Normal light-switch control returns immediately.
  • If the switch is in OFF, the pop-ups go DOWN.
  • If the switch is in HEAD, the pop-ups go UP.

Possible States

The rotary control provides seven fixed partial-open targets. Each step raises the pop-ups farther than the one before it.

Automatic Calibration

The controller learns timing during normal use so it can target the selected sleepy-eye position more consistently.

What the controller uses

The controller combines stored timing calibration data with live battery-voltage measurements to estimate how long each pop-up should move to reach the selected intermediate target.

This matters because pop-up travel time changes with supply voltage, so a fixed timer would not land in the same place under every condition.

What you need to do

For most users, no separate calibration routine is needed. The firmware continuously builds timing calibration data during normal use and stores it in non-volatile memory, so that learning survives power cycles.

Practical result

As the controller gathers more timing data, it can better match the intermediate sleepy-eye position on both sides.

Manual Adjustment

Use manual adjustment only when you need to fine-tune the right-hand side after the controller's automatic timing calibration has already had a chance to work.

OFFSET potentiometer

If the left and right pop-ups still do not stop at the same angle closely enough, use the controller's OFFSET potentiometer to fine-tune the RH pop-up timing.

Range

-50 ms to +50 ms

Center

0 ms

Applies To

Right-hand pop-up only.

Adjustment steps

  1. Select the sleepy-eye position you want to match.
  2. Turn Sleepy Eyes Mode on and compare the left and right pop-up angles.
  3. Make a small OFFSET adjustment.
  4. Turn the mode off and back on so the pop-ups move to the target again.
  5. Repeat in small steps until both sides match as closely as needed.

When to stop adjusting

If the mismatch is still too large after small OFFSET changes, keep using the controller normally so it can continue building timing calibration data before making larger manual adjustments.

Avoid chasing tiny differences

Make small changes and re-test each time. Large adjustment swings make it harder to tell whether the mismatch is improving or getting worse.
